## Model Download

The fine-tuned RoBERTa model is not included in this repo due to GitHub's file size limits.

**Download from Google Drive:** [\[Click here\](YOUR_GOOGLE_DRIVE_LINK)](https://drive.google.com/drive/folders/11ulkQ9fCmWuK8lC6InIT5U5npwjW9_I-?usp=sharing)

After downloading, place the following files inside a folder named `saved_model/` in the root of this project:

- model.safetensors
- config.json
- tokenizer.json
- tokenizer_config.json
- training_args.bin
